# Nihanshi Goyal Portfolio Website

A modern personal portfolio website built using React, TypeScript, Tailwind CSS, and Vite.

## Features

* Responsive modern UI
* Smooth animations
* Project showcase section
* Skills and experience section
* Contact section
* Fast performance using Vite
* Fully customizable

---

# Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite

---

# Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/NihanshiGoyal/Portfolio-website.git
```

---

## 2. Navigate to Project Folder

```bash
cd Portfolio-website
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

The project will run locally at:

```bash
http://localhost:5173
```

---

# Build for Production

```bash
npm run build
```

---

# Preview Production Build

```bash
npm run preview
```
